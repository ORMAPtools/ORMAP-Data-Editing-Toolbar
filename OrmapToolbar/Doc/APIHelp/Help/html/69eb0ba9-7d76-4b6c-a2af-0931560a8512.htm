This help file is built using Sandcastle Help File Builder (SHFB), a tool built by Eric Woodruff to make creating help files with Sandcastle easier. Sandcastle, created by Microsoft, is a tool used for creating MSDN-style documentation from .NET assemblies and their associated XML comments files.

Sandcastle Details

Sandcastle is a documentation compiler for Managed class library that generates Microsoft-style Help topics, both conceptual and API reference. It creates API reference topics by combining the XML documentation comments (/// in C# or ''' in VB) that are embedded in your source code with the syntax and structure of the types, which it acquires from reflecting against the associated .NET Framework assembly. Skeletal topics are created by using reflection on your project assembly (.dll) file. Additional content, such as remarks and parameter descriptions, is derived from text in the /// comments in your C# source code or ''' comments in your VB source code. Conceptual topics are created by converting XML documents that you author.

Sandcastle has been used by Microsoft to build their own documentation, such as the Visual Studio 2010 Beta 1 documentation.

Sandcastle Help File Builder (SFHB) Details

Sandcastle is command-line-based and has no GUI front-end, project management features, or an automated build process -- features many developers are used to from NDoc .

The Sandcastle Help File Builder was created to fill in the gaps, provide the missing NDoc-like features that are used most often, and provide graphical and command-line-based tools to build a help file in an automated fashion.

Example Installation Steps

Note

These are the steps I took to install Sandcastle Help File Builder and supporting apps and extensions.

-- Nick Seigal, Thursday, March 26, 2009

  1. Installed Microsoft Visual Studio 2005 SDK Version 4.0. Including:

    • Microsoft Visual Studio 2005 SDK Documentation
    • Innovasys Help Studio Lite

  2. Installed HTML Help Workshop (4.74.8703).

  3. Installed Sandcastle May 2008 (2.4.10520).

  4. Performed Sandcastle DXROOT fixes as described in SFHB Installation Instructions referenced above.

  5. Added Sandcastle Styles May 2008 Patch Rev 2 files to Sandcastle (copied files to %Program Files%\Sandcastle).

  6. Installed Sandcastle Help File Builder (1.7.0).

  7. Added Sandcastle Extensions SHFB Edition files to SHFB (copied files to %Program Files%\EWSoftware\Sandcastle Help File Builder).
